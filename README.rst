Features
--------

- Provides robust stitching of tiled microscope images, following algorithm by MIST_.
- Supports irregular-shaped tiles, if they are connected.

Installation
------------

You can install *M2Stitch* via pip_ from PyPI_:

.. code:: console

   $ pip install m2stitch
